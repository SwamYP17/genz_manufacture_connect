# GenZ Connects

GenZ Connects is a B2B application designed to revolutionize manufacturing entrepreneurship by integrating AI-driven analytics for personalized business insights and growth opportunities. This platform provides manufacturers with essential tools to streamline operations, analyze data efficiently, and expand their business potential.

## üì¶ Installation

Follow these steps to set up the project locally:

### Prerequisites
Ensure you have the following installed:
- Node.js & npm
- React Native CLI
- Flutter SDK (for mobile development)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/genz-connects.git
   cd genz-connects
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   - **For Web**:
     ```bash
     npm start
     ```
   - **For Android (React Native)**:
     ```bash
     npx react-native run-android
     ```
   - **For Flutter (Mobile Development)**:
     ```bash
     flutter run
     ```
