
# GenZ Connects - Manufacturing Platform

## About

GenZ Connects is a platform for connecting manufacturers and businesses.

## Features

- User authentication
- Industry exploration
- Stock management
- Product catalog
- Financial analytics
- Workflow management
- Profile management

## Development

This project is built with React, TypeScript, and Tailwind CSS.
